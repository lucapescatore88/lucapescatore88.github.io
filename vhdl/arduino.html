Fun with Arduino!


Arduino is a fantastic little tool which allows you to do simple electronics. It's a programmable board complete with pins an even coms with LEDs, capacitors, motors and other small electronic deviced with which you can have fun. And it is an italian project! In this gallery I report just a couple of videos of little circuits I've made.

In development...

Button: yes a simple reliable button.

A button... there's buttons everywhere nowadays. But it's not trivial at all to build a good reliable button. This project made me realise how amazing all electronics around us is.

A light that automatically switches on in the dark

In this project I use a resistance which is sensitive to light to automatically switch on a led when it's dark.

A shaded night light

This project simply complicated a bit the one before. Instead of using the light-sensitive resistance as a switch we can make the intensity of the led dependent on the amount of light.

Remote controlled light

In this project I use an infrared sensor to switch on a led. It works with any remote control!

Some led mess

In this project I put together some of what I learned in the previous ones. A button controls 4 leds. Pushing the button makes a white led increase intensity. After 3 pushes the white led switches off and 3 coloured leds switch on and off in a light pattern.

Raspberry Pi!

DC motors and servo motors

Here explained the difference between a DC motor and a servo motor. Both driven thanks to a variable resistance via an Arduino and a Raspberry Pi.
