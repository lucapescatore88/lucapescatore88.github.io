VHDL and FPGAs


FPGAs are Field Programmable Gate Arrays. They are sets of gates that can be connected to form a custom circuit. This allows to build ASIC (Application Specific Integrated Circuit) simply programming the connections. This is done via HDL (Hardware Description Language) languages. For these examples I use VHDL and an Altera FPGA board.
